Nabāhah (نباهة) – AI-based Laboratory Safety System

Nabāhah is an AI-powered laboratory safety monitoring system designed to enhance safety compliance and reduce real-world risks in laboratory environments. The platform uses computer vision to monitor PPE compliance and detect hazardous situations in real time, then provides actionable insights through an interactive dashboard and alerts.

Key features:
• Real-time detection of PPE violations, red-zone breaches, and chemical or liquid spills
• Live monitoring dashboard built with FastAPI
• Centralized logging of detections and alerts in Supabase
• Bilingual voice alerts (Arabic/English) using Edge-TTS
• RAG-based chatbot for laboratory safety queries

King County – Housing Sales Analytics

King County Housing Sales Analysis & Prediction is a data analytics and machine learning project that analyzes house sales data in King County (USA) and builds a predictive model for estimating housing prices. The project integrates exploratory data analysis, regression modeling, and an interactive dashboard to deliver actionable insights.

Key features:
• Exploratory data analysis (EDA) of housing prices and key property attributes
• Feature engineering with log-transformed house prices
• OLS regression model using key features such as living area (sqft), grade, bedrooms, bathrooms, condition, and waterfrontbr> • Model evaluation using R², RMSE, MAE, and accuracy metrics

InterviewSense – AI Interview Evaluation

InterviewSense is an AI-powered system that evaluates job applicants by analyzing speech, text, and facial expressions to provide objective assessments. It combines speech-to-text conversion, text similarity analysis, and facial emotion recognition to support fair and data-driven hiring decisions.

Key features:

• Q&A generation for interview flow support
• Text similarity scoring (Cosine & Jaccard)
